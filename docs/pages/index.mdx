# Introduction

Welcome to Extcy! Bringing the best Elixir utilities and functions to JavaScript and TypeScript.

## What is Extcy?

Extcy is a [NPM](https://www.npmjs.com/) package that provides a set of utilities for JavaScript and TypeScript. It is inspired by the [Elixir](https://elixir-lang.org/) programming language. Such as pattern matching, piping, Enums and more.

## Installation

You can easily get started by simply installing Extcy with just one simple command :)

```bash
npm install extcy
yarn add extcy
```

And now you are ready to take a look at our examples or guide to get started!
